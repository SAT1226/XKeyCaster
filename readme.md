# XKeyCaster
 keystroke visualizer for Linux.    
 ![image](./screen01.gif)
 
## Keyboard shortcuts
|Keys|Description|
| ---- | ---- |
|Left click + drag| Move window|
|Right click| Open Menu|

## System Requirements
* Linux

## Dependencies
* Qt 5.?

## Installation
```
$ cmake .
$ make
$ sudo make install
```
